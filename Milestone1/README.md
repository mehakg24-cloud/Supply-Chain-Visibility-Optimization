 # Supply Chain Visibility & Optimization - Milestone 1

## Project Overview
This project focuses on transforming a supply chain dataset into a star schema using Power BI.

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- GitHub

## Files Included
- Power BI Project (.pbix)
- Dataset
- Data Model Screenshot
- Documentation

## Star Schema
The project contains:
- Fact_table
- Dim_Customer
- Dim_Product
- Dim_Category
- Dim_Shipping
- Dim_Location
- Dim_Department
- Dim_Date
